Guide

Calculating the Basal Metabolic Rate (BMR) using the revised Harris-Benedict equation



* Men BMR = 88.362 + (13.397 x weight in kg) + (4.799 x height in cm) – (5.677 x age in years)



* Women BMR = 447.593 + (9.247 x weight in kg) + (3.098 x height in cm) – (4.330 x age in years)



1. Little to no exercise BMR x 1.2

2. Light exercise (1–3 days per week) BMR x 1.375

3. Moderate exercise (3–5 days per week) BMR x 1.55

4. Heavy exercise (6–7 days per week) BMR x 1.725

5. Very heavy exercise (twice per day, extra heavy workouts) BMR x 1.9
